# LumeAi Management Center

A modern, glassmorphic WebUI for managing the LumeAi Proxy Platform.

Designed by **Be1lnash3**.

## Features

- **Dashboard**: Real-time monitoring of model usage and system status.
- **Config Management**: Easy edition of configuration files.
- **Credential Management**: Upload and manage auth credentials securely.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
