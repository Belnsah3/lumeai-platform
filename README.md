# LumeAi.ru

LumeAi.ru - Advanced AI Proxy & Management Platform.

## Features

- **AI Proxy**: High-performance proxy for AI models.
- **Management UI**: Modern, glassmorphic web interface for administration.
- **Provider Support**: Integrated support for GitHub Copilot, LumeAI, and more.

## Quick Start

1. Configure `config.yaml` based on `config.example.yaml`.
2. Run the service:
   ```bash
   ./cli-proxy-api-plus
   ```
3. Access the dashboard at `https://panel.lumeai.ru`.

## Development

### Frontend
located in `lumeai-management-center/`

```bash
cd lumeai-management-center
npm install
npm run dev
```
